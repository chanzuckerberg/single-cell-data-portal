# Downloading Published Data on CZ CELLxGENE Discover

Clicking the download button launches a dialog that enables a dataset to be downloaded in h5ad (AnnData v0.8) and rds (Seurat v4) formats. All datasets adhere to the CELLxGENE single cell annotated data schema. Datasets can be downloaded via the browser by clicking the blue download button. The permanent download link can also be copied, shared, and pasted into a browser address bar.
